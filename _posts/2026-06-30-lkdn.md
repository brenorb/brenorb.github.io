**lkdn** is a **minimal CLI for supported LinkedIn API workflows**.

The scope is deliberately narrow: publish and manage **text, article, image, multi-image, poll, video, and document posts** through LinkedIn's official APIs, inspect or reuse uploaded media assets, and handle adjacent workflows such as **comments**, **reactions**, **social metadata**, and organization access preflight.

The project is opinionated about its boundaries. It does not try to automate OAuth in a fragile way, and it does not scrape LinkedIn pages or rely on undocumented web endpoints.

The result is a small tool that stays readable and practical while still covering the workflows that matter when you want to post, inspect, or edit LinkedIn content from the terminal.

The first published tag was **`v0.1.0` on 2026-06-30**, which is the date used for this project entry.
