**Freedom Skills** is a repository of **AI agent skills for Bitcoin, Nostr, and other freedom technologies**.

The goal is to help agents use tools for **uncensorable payments** and **censorship-resistant communication** more reliably, especially in contexts where depending on centralized services is a liability.

The project emphasizes reusable skills that can be tested, benchmarked, and shared, so freedom tech becomes more legible and more practical for autonomous systems.

The longer-term direction is a trusted layer for skill discovery, validation, and reuse in adversarial environments.
