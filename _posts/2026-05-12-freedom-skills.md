**Freedom Skills** is a repository of **AI agent skills for Bitcoin, Nostr, and other freedom technologies** that I started to make these tools more usable by autonomous systems.

I care about a boring but important gap here: many of the tools that matter for **uncensorable payments** and **censorship-resistant communication** are powerful, but still awkward for agents to use reliably.

I want the skills in this repo to be reusable, testable, benchmarkable, and shareable, so freedom tech becomes more legible and more practical for autonomous systems.

Longer term, I see Freedom Skills as a trusted layer for skill discovery, validation, and reuse in adversarial environments.
